# System
* 代表系统,系统级的很多属性和方都放置在此内部
* 构造器是private,成员变量和方法是static
* 成员变量  
    > in, out , err
* 成员方法
    > currentTimeMillis(), exit, gc, getProperty(String key)

## getProperty(String key)

|属性名|属性说明|
|:---|:---|
|java.version|java运行时环境版本|
|java.home|java安装目录|
|os.name|操作系统名称|
|os.version|操作系统版本|
|user.name|用户账户名称|
|user.home|用户的目录|
|user.dir|用户的当前工作目录|
